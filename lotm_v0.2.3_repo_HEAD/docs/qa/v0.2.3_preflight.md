# v0.2.3 Preflight Gate (no-deps) — Prospects MVP

**Last Updated:** 2026-02-14  
**Owner:** Dev Agent B (Tooling/QA/Docs)

## Command
- `npm run preflight`

This script is **no-deps compatible** (runs via `dist_batch/` and Node built-ins).

## What it checks (LOCKED minimum set)
1) ProspectsWindow schema snapshot (v1)
2) Determinism smoke (same seed/decisions ⇒ same ProspectsWindow ids + fields)
3) Non-perturbation when no prospects accepted (golden seeds; baseline captured from v0.2.2)

## Artifacts
Writes:
- `qa_artifacts/<APP_VERSION>_preflight.json`

## Baseline file (non-perturbation)
- `docs/qa/v0.2.3_non_perturbation_baseline_v0.2.2.json`

If golden seeds change, regenerate the baseline **only if the contract allows it**; otherwise keep seeds stable.

## Notes
- Preflight will **FAIL** if it cannot find any ProspectsWindow objects (schema_version = `prospects_window_v1`)
  during the scan window. This is intentional to prevent “Prospects not wired” regressions.
